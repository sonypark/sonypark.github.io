# 정규표현식(Regular Expression)

---

- 문자열의 특정 패턴을 찾을 수 있는 문법
- 패턴을 찾아서 추출, 삭제, 치환 등 문자열을 조작 할 수 있다.
- 실제 문자열 조작은 정규표현식을 알고 있는가 그렇지 않은가로 큰 차이를 보인다.

## 실무에서의 사용

- 이메일, 주소, 전화번호, 우편번호, 비밀번호 규칙 검증
- 입력값 중 불필요한 입력값 추출
- 트랜스파일링
- 개발 도구에서의 문자열 치환


### 우편번호 검사

```javascript
"123-123".match(/\d\d\d-\d\d\d/);
// [ '123-123', index: 0, input: '123-123', groups: undefined ]
"123-123".match(/\d\d\d-\d\d\d/)[0];
// 123-123
"123-123".match(/\d{3}-\d{3}/)[0]; // {} 안에 갯수를 적어줄 수 있다.
// 123-123
"12-123".match(/\d{3}-\d{3}/)[0]; // {} 안의 갯수가 안 맞으면 match가 되지 않는다.
// null


"12345".match(/\d{5}/)[0]; // 연속된 5자리수
// 12345

"12345".match(/(\d{3}-\d{3}|\d{5})/)[0] // () 안에 | 연산자를 쓰면 OR 연산자가 된다.

// [] 안에 범위를 지정해줄수있다.
"82345".match(/(\d{3}-\d{3}|[012345678]\d{4})/)[0] // 앞에 9가 오면 안되는 경우
// [0-8] 연속된 숫자의 경우 - 를 사용해 표현할 수 있다.
"82345".match(/(\d{3}-\d{3}|[0-8]\d{4})/)[0] // 앞에 9가 오면 안되는 경우
// [0-46-8] 중간에 있는 숫자도 이런식으로 제외할 수 있다.
"82345".match(/(\d{3}-\d{3}|[0-46-8]\d{4})/)[0] // 앞에 5가 오면 안되는 경우


// 알파벳의 경우
/[a-zA-Z]/
/[a-fh-k]/
```

### 휴대폰 번호

```javascript
// 휴대폰 번호
console.log("010-3432-1235".match(/01[01789]-\d{3,4}-\d{4}/)[0]);

// 01[01789] 첫자리는 010,011,017,018,019 만 가능하도록 표현
// \d{3,4} 두번째 자리는 3~4자리 가능하도록 표현. {}안에 콤마(,)를 이용해 범위 지정
```
