# 백준 2193 - 이친수 (DP) {docsify-ignore-all}

> 출처: https://www.acmicpc.net/problem/2193

## 문제

0과 1로만 이루어진 수를 이진수라 한다. 이러한 이진수 중 특별한 성질을 갖는 것들이 있는데, 이들을 이친수(pinary number)라 한다. 이친수는 다음의 성질을 만족한다.

1. 이친수는 0으로 시작하지 않는다.
2. 이친수에서는 1이 두 번 연속으로 나타나지 않는다. 즉, 11을 부분 문자열로 갖지 않는다.

예를 들면 1, 10, 100, 101, 1000, 1001 등이 이친수가 된다. 하지만 0010101이나 101101은 각각 1, 2번 규칙에 위배되므로 이친수가 아니다.

N(1 ≤ N ≤ 90)이 주어졌을 때, N자리 이친수의 개수를 구하는 프로그램을 작성하시오.

#### 입력

첫째 줄에 N이 주어진다.

#### 출력

첫째 줄에 N자리 이친수의 개수를 출력한다.

#### 예제 입력1

```
3
```

#### 예제 출력1

```
2
```

## 접근 방법

- 직접 써보면서 규칙을 찾아보니, n번째 이친수의 개수는 1부터 n-2번째 이친수의 개수의 합과 같았다.

- 그래서 그대로 식을 만들어 풀었다.



## 내 풀이

```python
n = int(input())

dp = [0,1,1,2,3]

def dinary(k):
    return sum(dp[:k-1])

for i in range(5,n+1):
    dp.append(1+dinary(i))

print(dp[n])
```

- 풀고나서 다른 사람의 풀이를 보니, 결국은 피보나치 수열이란 걸 깨달았다.

## 다른 사람 풀이

- 이친수의 조건에 따라 다음 규칙이 성립된다.

- **이친수의 조건: 0으로 시작할 수 없고 1이 연속해서 오면 안 된다.**

- 따라서 **이친수**는 (두 자리 이상일 때) **항상 `10`으로 시작**한다.

- `10`으로 시작하고 그 다음에 오는 수는 `1 또는 0` 이다.

    - 그 다음 수로 `1`이 오는 경우는 `n-1 번째` 이친수의 개수와 같다.

    - 그 다음 수로 `2`가 오는 경우는 `n-2 번째` 이친수의 개수와 같다.

- 즉, n번째 이친수의 개수는 `n-1번째` 이친수의 개수와 `n-2번째` 이친수의 개수의 합과 같다.

    - 결국 피보나치 수열..!

```python
N = int(input())
d = [1, 1]
for i in range(2, N+1):
    d.append(d[i-1] + d[i-2])
print(d[N-1])
```

## 느낀점
- 기본적으로 피보나치 수열과 같은 규칙이다. DP 문제는 피보나치 수열과 같거나 비슷한 규칙이 참 많다는 걸 느꼈다.
